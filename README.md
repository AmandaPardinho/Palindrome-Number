# Palindrome Numbers

### Abstract
>Palindrome number is a number reads the same backward as forward.

---
### The Problem
>Given an integer x, where 1 <= x <= 9999, you must: \
>* Return true if x is palindrome integer; \
>* Find all palindrome numbers.
