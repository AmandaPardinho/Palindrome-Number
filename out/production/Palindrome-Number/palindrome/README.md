# Palindrome Numbers

### Abstract

> Palindrome number is a number reads the same backward as forward.

---

### The Problem

> Given an integer x, where 1 <= x <= 9999, you must:
>
> * If true return x is palindrome integer;
> * If false return x is not a palindrome integer.
